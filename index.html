retro-counter

A little 90s hit counter for static sites. One file, no dependencies, no server, no third-party service, no tracking.

← this is real. Close the tab, come back, and it ticks up.

Read this part before you use it. The count lives in each visitor's own browser, so it is not a site-wide total. Every visitor counts only their own visits, and a first-time visitor always sees 000001. It's a personal hello, not a stat. That's why the default label reads "Your visit" rather than the period-accurate "You are visitor number" — that phrasing would be a lie. If you need a true shared tally you need a server or a hosted counter service, and this isn't that.

Install

Download retro-counter.js and drop it in your site. Then:

<div data-retro-counter></div>
<script src="retro-counter.js"></script>

That's the whole thing. The styles come with it — there's no CSS file to add.

Options

All optional, all set with data attributes:

<div data-retro-counter
     data-label="Your visit"
     data-digits="6"
     data-theme="lcd"
     data-key="my-site"
     data-count-per="session"></div>
AttributeDefaultWhat it does
data-labelYour visitText before the digits. Set to "" for digits only.
data-digits6How many digits. The number caps at the largest that fits.
data-themelcdlcd, amber, cyan, or paper.
data-keyretro-counterlocalStorage key. Change it if you run two separate counters.
data-count-persessionsession, pageview, or day.

Themes

lcd
amber
cyan
paper

They all share data-key="demo", so they show the same number and the visit is only counted once.

Making it yours

Override the CSS custom properties — no need to touch the file:

.rc {
  --rc-box:   #2c2420;  /* the odometer housing */
  --rc-digit: #9fc131;  /* digit color         */
  --rc-glow:  rgba(159, 193, 49, .75);
  --rc-label: #6a5f57;
}

Counting rules

  • session (default) — one count per browsing session, so clicking around your site doesn't run it up.
  • pageview — every page load.
  • day — at most one count per calendar day.

If storage is blocked (some private modes, or cookies off) the counter hides itself rather than showing a fake or stuck number.
